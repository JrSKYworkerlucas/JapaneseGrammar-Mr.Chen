# P20 <ruby>感<rt>かん</rt>情<rt>じょう</rt></ruby>形容(動)詞、<ruby>希<rt>き</rt>望<rt>ぼう</rt></ruby>
1. ～がる
2. ～たい
3. ～欲しい

## 1. ～がる

有一类形容词、形容动词是专门表示情感的，比如：
<ruby>嬉<rt>うれ</rt>しい</ruby>、
<ruby>悲<rt>かな</rt>しい</ruby>、
<ruby>怖<rt>こわ</rt>い</ruby>、
<ruby>不思議<rt>ふしぎ</rt>だ</ruby>···
被称为
<ruby>感<rt>かん</rt>情<rt>じょう</rt></ruby>・
<ruby>感<rt>かん</rt>覚<rt>かく</rt></ruby>
形容詞、形容動詞。

这类词语一般只能陈述第一人称的情感。如果要表达第二、第三人称的情感，要在词干后接～がる。
例如：嬉し**がる**、不思議**がる**。此时，变化后的词性为五段动词。

| 例文                       | MEMO                                                   |
| -------------------------- | ------------------------------------------------------ |
| 嬉しい。                   |                                                        |
| 嬉しい？                   | 高兴吗？up tone 可以用于询问第二人称的情感             |
| 佐藤さんは嬉しがっている。 | 因为感情是持续一段时间的，所以用～ている的方式进行表达 |

## 2. ～たい
～たい表示想进行某个动作、想要做...。使用方式是<u><b>連用形+たい</b></u>，也是只能用于表达第一人称想做的动作。如果表达其他人称需要后接～がる。
| 例文                               | MEMO |
| ---------------------------------- | ---- |
| リンゴが食べたい。                 |      |
| リンゴが食べたいですか。           |      |
| リンゴが食べたくない。             |      |
| 彼女は海外旅行に行きたがってます。 |      |

## 3. ～欲しい
～欲しい前面可以是名词，也可以是动作。同样只能修饰第一人称想要的东西，如果修饰第二、第三人称要后接～がる。

如果是名词+欲しい，一般是汉字；如果是动词 + ほしい，一般是平假名。

| 例文                                               | MEMO                                                                                           |
| -------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| 同じ趣味がある友達が欲しい。                       | 想要的是名词，不是动作。                                                                       |
| 母は大きい冷蔵庫をほしがっています。               | 母亲想要一个大大的冰箱，由于～がる导致变为名词，冰箱做为宾语，所以要用格助词を                 |
| 彼に早く<ruby>謝<rt>あやま</rt></ruby>ってほしい。 | 想让他快点道歉，～て+欲しい。</br>意思和たい相反，たい是自己想做什么，ほしい是想让别人做什么。 |
| 早く晴れてほしい。                                 | 想让天快点放晴。想让...前面也可以是抽象的概念，不一定是人                                      |
